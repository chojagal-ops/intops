🏭 1. 시스템 개요

INTOPS 설비점검 시스템은 공장 내 설비의 일상 점검을 디지털로 관리하고, 이상 발생 이력을 체계적으로 추적하는 플랫폼입니다.

🔑 3. 회원가입 및 로그인
1
회원가입 신청

로그인 화면 → 회원가입 신청 → 성명, 사번, 이메일, 비밀번호, 소속 팀 입력 후 신청합니다.

2
관리자 승인 대기

관리자가 신청을 확인하고 역할(점검자/승인자)을 지정합니다. 승인 완료 후 로그인이 가능합니다.

3
로그인

사번과 비밀번호를 입력하여 로그인합니다. 이메일은 이상 알림·비밀번호 찾기에 사용되므로 정확히 입력하세요.

4
비밀번호 분실 시

로그인 화면 → 비밀번호를 잊으셨나요? → 사번·이름 입력 → 등록된 이메일로 인증번호 발송. 이메일이 없으면 관리자에게 재설정 요청하세요.

🔍 4. 설비 점검 방법
1
점검 화면 접속

설비에 부착된 QR 코드를 스마트폰으로 스캔하거나, 상단 메뉴 월별 점검 결과에서 설비명을 클릭합니다.

2
점검 항목 입력

항목별로 정상 / 이상 / 수리필요 / 휴동을 선택합니다. 특이사항은 비고란에 입력합니다.

3
이상 항목 자동 감지

이상 또는 수리필요를 선택하면 이상 신고 패널이 자동으로 열립니다. 이상 내용을 바로 등록할 수 있습니다.

4
점검 완료 제출

점검 완료 버튼 클릭 → 기록 저장 + 승인자에게 이메일 자동 발송. 1일 1회 기준으로 집계됩니다.

💡 주1회·월1회 주기의 설비는 점검한 날 제출하면 해당 기간 나머지 평일이 자동으로 동일 결과로 채워집니다.
✅ 5. 점검 승인 (승인자)
1
승인 대기 확인

점검 제출 시 이메일 알림을 수신합니다. 홈(대시보드)의 승인 대기 목록에서도 확인 가능합니다.

2
승인 처리

대기 목록에서 ✔ 승인 버튼 클릭. 상세보기 버튼으로 점검표를 먼저 확인할 수도 있습니다.

3
승인 완료

승인된 점검은 승인완료 상태로 변경되며, 점검자에게도 이메일 알림이 발송됩니다.

💡 설비별 승인자는 관리자가 설비 관리에서 지정합니다. 승인자로 지정된 설비의 점검만 담당합니다.
📊 6. 점검 결과 조회
메뉴내용
내 점검 결과 내가 점검한 설비 목록과 결과를 날짜별로 조회합니다. 점검자는 자신의 기록만 조회 가능합니다.
일별 점검 결과 특정 날짜를 선택하여 전체 설비 점검 현황을 한눈에 조회합니다.
월별 점검 결과 설비별 월간 점검 현황을 달력 형식으로 조회합니다. 엑셀 다운로드 및 인쇄 기능을 제공합니다. 토·일요일은 자동으로 휴동(P) 표시됩니다.
점검율 모니터링 팀별·설비별 월간 점검율 통계를 차트와 표로 확인합니다. 엑셀로 내려받을 수 있습니다.
🚨 7. 이상발생관리 NEW
📝 이상 신고 방법
1
이상 감지 (자동 / 수동)

점검표에서 이상 또는 수리필요를 선택하면 이상 신고 패널이 자동으로 열립니다. 또는 설비 점검 화면 하단의 🚨 이상 신고 버튼을 클릭합니다.

2
이상 내용 입력

우선순위(긴급/높음/보통/낮음), 발생일시, 이상 내용(필수), 조치사항, 조치완료 예정일을 입력합니다.

3
조치담당자 지정 및 이메일 발송

조치담당자 이름을 입력하면 시스템에 등록된 해당 사용자에게 이메일 자동 발송됩니다. 입력란에 시스템 사용자 이름이 자동 완성으로 제안됩니다.

4
사진 첨부 (최대 5장)

이상 부위 사진을 최대 5장까지 첨부할 수 있습니다. 사진은 자동으로 압축 처리되어 업로드됩니다.

5
조치 완료 처리

조치가 완료되면 이상발생관리 대장에서 해당 이력의 조치완료를 선택하고 완료일을 입력합니다.

📋 이상발생관리 대장
기능설명
필터링팀(부서), 설비, 우선순위, 조치상태별로 이력을 필터링합니다.
📥 엑셀 다운로드현재 필터 조건의 이상 이력을 엑셀 파일로 내려받습니다. 우선순위별 색상 서식 포함.
🖨 프린트인쇄에 불필요한 UI를 숨기고 관리대장 형식으로 출력합니다.
사진 보기사진 아이콘 클릭 → 라이트박스로 확대 보기. ←/→ 키 또는 버튼으로 사진을 넘깁니다.
차트 분석하단에 월별 발생 추이, 팀별 현황, 우선순위 분포, 월별 완료율 차트를 제공합니다.
🛡 관리자 전용 기능
  • ✏️ 수정: 이상발생 내용(설비, 우선순위, 이상내용, 조치사항, 담당자, 날짜 등) 전체 수정. 기존 사진 삭제 및 새 사진 추가 가능.
  • 🗑 삭제: 이상 이력 및 연결된 사진 전체 삭제. 삭제 전 확인 팝업이 표시됩니다.
💡 이상 신고 시 긴급 우선순위로 지정하면 관리대장에서 빨간색으로 강조 표시됩니다. 미조치 긴급 건수는 통계 카드에서 별도로 확인할 수 있습니다.
🛠 8. 관리자 기능
👥 회원 관리
  • 신규 회원가입 신청을 승인/거부하고 역할(점검자/승인자)을 지정합니다.
  • 기존 회원의 역할 변경, 이메일·연락처 수정, 계정 비활성화가 가능합니다.
  • 회원 이메일이 정확해야 점검 승인 알림 및 이상발생 알림이 정상 발송됩니다.
⚙️ 설비 관리
  • 설비 등록: 설비명, 위치, 담당부서, 관리번호, 점검 주기, 담당자(정/부) 입력.
  • 승인자 지정: 드롭다운 선택 시 자동 저장.
  • 점검표 업로드: 엑셀 템플릿 업로드 → 점검 항목으로 사용.
  • QR 다운로드: 설비별 QR 이미지 다운로드 후 설비에 부착.
  • 설비 수정/삭제: 등록 정보 수정 또는 삭제 (삭제 시 관련 점검 기록도 삭제).
🗂 데이터 관리
  • 대시보드 → 관리자 도구 → 오늘 전체 휴동 처리: 당일 전체 설비를 일괄 휴동 처리합니다.
  • 데이터 관리 페이지에서 중복 점검 기록 정리, 날짜별 점검 삭제 등을 할 수 있습니다.
📧 9. 이메일 알림
알림 시점수신자내용
점검 제출 시 승인자 새 점검 결과 제출 알림 (설비명, 점검자, 결과, 링크 포함)
승인 처리 시 점검자 점검 결과 승인 완료 알림
이상발생 등록 시 NEW 조치담당자 이상발생 조치 요청 알림 (설비명, 우선순위, 이상내용, 예정일, 링크 포함)
평일 오전 11시 자동 미점검 설비 담당자(정/부) 해당 설비 미점검 알림 목록 자동 발송
📩 이메일이 오지 않으면 ① 내 정보에서 이메일 주소 확인 ② 스팸함 확인 ③ 관리자에게 SMTP 설정 문의.
❓ 10. 자주 묻는 질문 (FAQ)
로그인이 안 됩니다.
관리자 승인이 완료되어야 로그인할 수 있습니다. 비밀번호 분실 시 비밀번호를 잊으셨나요?를 이용하세요. 계속 문제가 있으면 관리자에게 문의하세요.
QR 코드 스캔 후 반응이 없습니다.
로그인 상태에서 스캔하세요. 미로그인 시 로그인 페이지로 이동합니다. 월별 점검 결과 메뉴에서 설비명을 클릭해도 점검표에 접속할 수 있습니다.
점검 횟수가 실제보다 적게 표시됩니다.
점검 횟수는 1일 1회 기준으로 집계됩니다. 같은 날 여러 번 제출해도 최신 1건으로 집계되므로 정상입니다.
이상 신고 시 이메일이 발송되지 않습니다.
① 조치담당자로 입력한 이름이 시스템에 등록된 사용자 이름과 정확히 일치해야 합니다. ② 해당 사용자의 이메일 주소가 내 정보에 등록되어 있어야 합니다. ③ 관리자에게 SMTP 설정 여부를 확인하세요.
이상발생관리 대장을 엑셀로 받으려면?
상단 메뉴 이상발생관리 → 필터 조건 선택(선택사항) → 📥 엑셀 다운로드 버튼 클릭. 현재 필터 조건이 적용된 데이터가 다운로드됩니다.
이상 이력을 수정하거나 삭제하려면?
관리자 계정으로 로그인하면 이상발생관리 대장에서 각 이력 우측의 ✏️ 수정 또는 🗑 삭제 버튼이 표시됩니다.
점검표 엑셀을 다운로드하려면?
월별 점검 결과 → 설비 선택 → 점검표 상단의 엑셀 다운로드 버튼 클릭.
설비를 추가하거나 삭제하려면?
관리자 계정으로 로그인 후 상단 메뉴 설비 관리에서 추가, 수정, 삭제할 수 있습니다.
추가 문의는 시스템 관리자에게 연락하세요.
