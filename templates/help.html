1. 시스템 개요

INTOPS 설비점검 시스템은 공장 내 설빔의 일상 점검을 디지털로 관리하는 플랫폼입니다.

①
설비 등록
관리자가 설비를 등록하고 승인자를 지정합니다.
→
②
QR 스캔
설비에 부착된 QR을 스캔하여 점검 화면에 접속합니다.
→
③
점검 입력
항목별 결과를 입력하고 제출합니다.
→
④
승인 처리
승인자가 결과를 확인하고 승인합니다.
2. 역할 안내
역할설명주요 기능
점검자설비 일상 점검 담당자QR 스캔 → 점검 입력 → 제출 / 내 점검 결과 조회
승인자점검 결과 검토·승인 담당자대시보드에서 승인 대기 건 확인 및 승인 처리
관리자시스템 최고 권한자회원 승인·역할 지정 / 설비 등록·수정·삭제 / QR 생성
3. 회원가입 및 로그인
1
회원가입 신청

로그인 화면 → 회원가입 신청 → 성명, 사번, 이메일, 비밀번호, 소속 팀 입력 후 신청.

2
관리자 승인 대기

관리자가 승인하고 역할(점검자/승인자)을 지정합니다. 승인 후 로그인 가능.

3
로그인

사번과 비밀번호를 입력하여 로그인합니다.

4
비밀번호 분실 시

로그인 화면 → 비밀번호를 잊으셨나요? → 사번·이름 입력 → 이메일로 인증번호 발송.

4. 설비 점검 방법
1
QR 코드 스캔

설비에 부착된 QR 코드를 스마트폰으로 스캔합니다. 또는 월별 점검 결과 메뉴에서 설비를 클릭해도 됩니다.

2
점검 결과 입력

항목별로 정상 / 이상 / 수리필요 / 휴동을 선택하고 특이사항은 비고란에 입력합니다.

3
점검 완료 제출

점검 완료 버튼 클릭 → 기록 저장 + 승인자에게 이메일 자동 발송.

4
1일 1회 기준

같은 날 중복 제출해도 1일 1회로만 집계됩니다.

5. 점검 승인 (승인자)
1
승인 대기 확인

점검 제출 시 이메일 알림 수신. 대시보드에서 승인 대기 목록 확인.

2
승인 처리

대기 목록에서 ✔ 승인 버튼 클릭. 상세보기로 먼저 확인 가능.

3
승인 완료

승인된 점검은 승인완료 상태로 변경됩니다.

💡 설비별 승인자는 관리자가 설비 관리에서 지정합니다.
6. 점검 결과 조회
메뉴내용
내 점검 결과내가 점검한 설비 목록과 결과를 날짜별로 조회합니다.
일별 점검 결과특정 날짜를 선택하여 전체 설비 점검 현황을 조회합니다.
월별 점검 결과설비별 월간 점검 현황을 캘린더 형식으로 조회하고 엑셀로 다운로드합니다.
📅 토·일요일은 월별 점검표에서 자동으로 휴동(P)으로 표시됩니다.
7. 관리자 기능
👤 회원 관리
  • 신규 회원가입 신청을 승인/거부하고 역할(점검자/승인자)을 지정합니다.
  • 기존 회원의 역할 변경 및 계정 비활성화가 가능합니다.
⚙️ 설비 관리
  • 설비 등록: 설비명, 위치, 담당부서, 관리번호, 담당자(정/부) 입력.
  • 승인자 지정: 드롭다운 선택 시 자동 저장.
  • 점검표 업로드: 엑셀 템플릿 업로드 → 점검 항목으로 사용.
  • QR 다운로드: 설비별 QR 이미지 다운로드 후 설비에 부착.
  • 설비 수정/삭제: 등록 정보 수정 또는 삭제.
🗑 중복 점검 기록 정리
  • 대시보드 → 관리자 도구 → 중복 점검 기록 일괄 정리: 날짜·설비별 최신 1건만 남기고 삭제.

9. 자주 묻는 질문 (FAQ)
Q. 로그인이 안 됩니다.
관리자 승인이 필요합니다. 비밀번호 분실 시 비밀번호를 잊으셨나요?를 이용하세요.
Q. QR 코드 스캔 후 반응이 없습니다.
로그인 상태에서 스캔하세요. 미로그인 시 로그인 페이지로 이동합니다. 월별 점검 결과에서 직접 클릭해도 됩니다.
Q. 점검횟수가 실제보다 적게 표시됩니다.
점검횟수는 1일 1회 기준으로 집계됩니다. 같은 날 여러 번 제출해도 1회로만 집계되므로 정상입니다.
Q. 이메일 알림이 오지 않습니다.
등록된 이메일 주소를 확인하고 스팸함도 확인해보세요. 계속 문제가 있으면 관리자에게 문의하세요.
Q. 점검표 엑셀을 다운로드하려면?
월별 점검 결과 → 설비 선택 → 엑셀 다운로드 버튼 클릭.
Q. 설비를 추가하거나 삭제하려면?
관리자 계정으로 로그인 후 설비 관리 메뉴에서 추가, 수정, 삭제할 수 있습니다.
추가 문의는 시스템 관리자에게 연락하세요.
