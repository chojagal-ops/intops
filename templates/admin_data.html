데이터 관리
점검 기록 관리 및 데이터 백업을 처리합니다.
🗑 월별 점검 기록 삭제
선택한 달의 모든 점검 기록(승인 완료 포함)을 완전히 삭제합니다.
⚠ 이 작업은 되돌릴 수 없습니다. 삭제 전 백업을 권장합니다.
📅 일별 점검 기록 삭제
선택한 날짜의 모든 점검 기록(승인 완료 포함)을 완전히 삭제합니다.
⚠ 이 작업은 되돌릴 수 없습니다.
⏸ 날짜별 전 설비 휴동 처리
선택한 날짜의 기존 점검 기록을 삭제하고,
모든 설비에 휴동(승인완료) 기록을 자동 생성합니다. (공휴일·휴무일 등)
🔧 중복 점검 기록 정리
설비·날짜별로 중복된 점검 기록을 정리합니다.
각 날짜마다 최신 1건만 남기고 나머지 중복 기록은 삭제됩니다.
🗄️ 데이터 백업
전체 데이터베이스(사용자·설비·점검기록 등)를 CSV ZIP 파일로 다운로드합니다.
정기적으로 백업해두면 데이터 손실 시 복구가 가능합니다.
⬇️ DB 전체 백업 다운로드 intops_backup_YYYYMMDD_HHMMSS.zip 형태로 저장됩니다
