📅 일별 점검 기록 삭제

⏸ 날짜별 전 설비 휴동 처리
선택한 날짜의 기존 점검 기록을 삭제하고,
모든 설비에 휴동(승인완료) 기록을 자동 생성합니다. (공휴일·휴무일 등)
🔧 중복 점검 기록 정리

각 날짜마다 최신 1건만 남기고 나머지 중복 기록은 삭제됩니다.

정기적으로 백업해두면 데이터 손실 시 복구가 가능합니다.
⬇️ DB 전체 백업 다운로드 intops_backup_YYYYMMDD_HHMMSS.zip 형태로 저장됩니다
